# Resource Tracker PWA

A Progressive Web App for tracking investment resources including gold, cryptocurrency, stocks, and real estate.

## Features

- **Multi-Resource Support**: Track gold (SJC, DOJI, PNJ), cryptocurrencies, stocks, and houses
- **Real-time Pricing**: Auto-fetch gold prices from VN API and crypto prices from Binance
- **Buy/Sell Tracking**: Record purchases and sales with profit calculations
- **Data Management**: Export/import data for backup and migration
- **PWA Ready**: Install as mobile app with offline support

## Live Demo

Access the app at: `https://[your-username].github.io/resource-tracker`

## Local Development

```bash
npm install
npm run dev
```
